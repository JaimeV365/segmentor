# FAQ Content Management

This directory contains all FAQ questions and answers in markdown format for easy maintenance and updates.

## Structure

- `README.md` - This file (overview)
- `getting-started.md` - Getting Started questions
- `using-tool.md` - Using the Tool questions
- `saving-exporting.md` - Saving & Exporting questions
- `customisation.md` - Customisation & Branding questions
- `privacy-security.md` - Privacy & Security questions
- `understanding-model.md` - Understanding the Model questions
- `troubleshooting.md` - Troubleshooting questions

## Format

Each FAQ entry follows this format:

```markdown
### Question Title
**Keywords:** keyword1, keyword2, keyword3

Answer content here. Keep it short and sharp. Use British English.

- Bullet points for lists
- Keep answers concise
```

## Category Mapping

- `getting-started` → Getting Started
- `using-tool` → Using the Tool  
- `saving-exporting` → Saving & Exporting
- `customisation` → Customisation & Branding
- `privacy-security` → Privacy & Security
- `understanding-model` → Understanding the Model
- `troubleshooting` → Troubleshooting (if needed)

## Maintenance

1. Add new questions to the appropriate category file
2. Update existing questions as features change
3. Keep answers short, sharp, and in British English
4. Use keywords for search functionality
5. When ready, these can be converted to HTML (manual or via script)

## Notes

- All content should be in British English
- Avoid using "Premium" - use "Brand+" instead
- Keep answers concise and actionable
- Include links where relevant (e.g., to upgrade pages)
